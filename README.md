# Whatley National Trust - Crypto & Digital Banking Platform

## 🏦 Project Overview

A comprehensive digital banking platform featuring cryptocurrency integration and luxury aviation business ventures. This project showcases the Whatley National Trust banking services and the revolutionary Whatley's AirWay business plan.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start the development server
npm start

# Server will be available at http://localhost:3000
```
